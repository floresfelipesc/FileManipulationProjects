DESCRIPTION:
This is a Console App that given a folder's filepath, will sort all image files
that are lower in resolution than 1920 x 1080 into a subfolder. This does
not include subfolders of the folder you provide.

HOW TO INSTALL: 
1. Navigate into the "Executable" Folder
2. Click the green "Code" button
3. Download ZIP
4. Navigate to the Zip file on your system
5. Extract Zip
6. Run the .exe file

HOW TO USE: 
1. Run the .exe file
2. Provide the filepath for a folder on your computer
3. Check the folder you input 
4. Look at subfolder named "excluded"
5. These are the images that are not greater than or equal to 1920 x 1080 resolution
