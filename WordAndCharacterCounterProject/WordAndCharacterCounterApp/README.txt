DESCRIPTION:
This is a Console App that given a piece of text within the "input.txt" 
file within this folder, will count and display the number of words and characters 
of that piece of text. 

HOW TO INSTALL: 
1. Navigate into the "Executable" Folder
2. Click the green "Code" button
3. Download ZIP
4. Navigate to the Zip file on your system
5. Extract Zip
6. Run the .exe file

HOW TO USE: 
1. Copy your desired text
2. Paste desired text into "input.txt" file
3. Run the .exe file
4. The UI displays the number of words and characters 
(with and without counting spaces)
